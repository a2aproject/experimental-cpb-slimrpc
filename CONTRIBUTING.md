# Contributing

This repository follows the same contribution guidelines as the main A2A project.

Please refer to [CONTRIBUTING.md in the a2aproject/A2A repository](https://github.com/a2aproject/A2A/blob/main/CONTRIBUTING.md) for full details on:

- Development setup and prerequisites
- Working with documentation
- Code standards (linting, formatting, conventional commits)
- The contribution workflow (fork, branch, PR process)

## Code of Conduct

This project also follows the A2A [Code of Conduct](https://github.com/a2aproject/A2A/blob/main/CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code.
